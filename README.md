
# MusicGear- The Music Production Equipment

A modern e-commerce website for music production equipment built using HTML, CSS, and JavaScript. It features product categories, dynamic product display, login/signup system, profile management, liked products, cart, and a checkout system with simulated UPI/card payment, offering a smooth user experience.

## 🚀 Features

Fully Responsive Design (Mobile + Desktop)  
Category-Based Product Browsing  
Search Functionality  
Dynamic Cart System (using localStorage)  
Login/Signup & Profile System  
Liked Products (Wishlist Feature)  
Order System with My Orders section  
Payment Simulation (UPI / Card)  
Login Required for Actions (Cart / Like)  
Modern UI with Animations & Hover Effects

## 🎯 How It Works

Users can browse different music equipment categories  
Each category loads products dynamically  
Users can view product details using Quick View  
Login is required to like products or add to cart  
Cart stores selected items using localStorage  
Checkout simulates real payment methods like UPI/Card  
Orders are saved and displayed in My Orders section
